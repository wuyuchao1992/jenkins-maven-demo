4. 能够使用K8s查看日志、排查测试环境各类问题

5. 一年以上JMeter性能测试实操经验，可独立开展测试、分析性能瓶颈

6. 熟悉服务端常用技术，掌握Redis、MongoDB、Kafka、MQ、MySQL、Nginx等常用技术组件

7. 具备基础产品体验意识，可识别产品交互与页面各类可用性问题
英文版 Requirements（精简最终版）

1. 3+ years of automation test development experience, proficient in Cucumber + Selenium and backend API automation.

3. Familiar with Maven, Git and CI/CD workflow.

4. Able to check logs and troubleshoot environment issues with Kubernetes.
